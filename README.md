# Sales Web App

## Getting Started

Install dependencies:

```shell
yarn
```

Run the server:

```shell
yarn dev
```

Develop your feature. 👩🏼‍💻

Test it!

```shell
yarn test
```

Commit your changes:

```shell
yarn commit
```

Enjoy!🎉
